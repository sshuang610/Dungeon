# Dungeon Adventure Game

A terminal-based RPG built to practice Object-Oriented Programming (OOP) principles.

## Features
* **Room-based Navigation:** Explore a map built with custom data structures.
* **OOP Design:** Implements polymorphism and inheritance for game entities.
* **Combat System:** Engage in battles with NPCs and manage inventory.

## Tech Stack
* **Language:** C++
* **Tools:** Makefile

## How to Run
1. Compile: `make`
2. Run: `./dungeon`
